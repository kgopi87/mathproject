Welcome to REST Calculator

The Math Application Peripheral Interface is used for generic match caculation using Python.
The purpose of the project is to design simple Math Calculator using REST implementation using HTTP protocal

Use the project for testing purpose and Please don't run the load testing here.

 API Documentation        

 /auth

The /auth purpose is to generate a token and display in UI as rendered HTML.

Request:

  • Resource/path: /auth
  • Method: GET

Response:

Example: response with generated token

bMmw1VDdaNTRxNzlZVmpHbTAyYVAxWThHSjU1MDAzcFd6TGNOS

/maths/v1/add

The /maths/v1/add purpose is to add two number and provide the results with addition of two numbers. The API has two request method for doing Addition both the methods provide the exact same values.

**GET**

Request:

  • Resource/path: /maths/v1/add?id1=4&id2=8
  • Method: GET
  • Query Paramters: id1, id2
  • Header: Auth: token

Response:

{

"added": 12,

"id1": 4,

"id2": 8

}

 

**POST**

Request:

  • Resource/path: /maths/v1/add
  • Method: POST
  • Media Type: application/json
  • Body: { "id1":2, "id2":4}
  • Header: Auth: token

Response:

{

"added": 6

}
